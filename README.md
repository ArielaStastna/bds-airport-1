Steps for building the project using command line: 1) git clone https://github.com/ArielaStastna/bds-airport-1/
                                                   2) cd bds-airport-1 
                                                   3) mvn clean install
                                                   4) cd target
                                                   5) java -jar bds-airport-1.0.0.jar
Project is using Java version 11.0.12 and Maven version 3.8.4.
